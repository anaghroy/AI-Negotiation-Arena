AI Negotiation Arena

An interactive full-stack web application where users negotiate with an AI seller to purchase a product at the lowest possible price. The game simulates real-world bargaining using multi-round conversations, strategy, and decision-making. Your final deal determines your position on a global leaderboard.

🎯 Features

🧠 Core Gameplay
Multi-round negotiation with an AI seller
Dynamic price updates based on user input
AI personality (stubborn, neutral, motivated)
Smart negotiation hints

🏆 Leaderboard
Global leaderboard based on lowest deal price
Top 3 players highlighted with premium UI
Real-time updates after each completed game

🔐 Authentication
Secure login & registration
JWT-based authentication
HTTP-only cookies for session management
Persistent login using Zustand

🎨 UI/UX
Chat-based negotiation interface
Sound effects (send, receive, success)
Animated leaderboard UI
Dropdown user profile with logout

🏗️ Tech Stack
Frontend
React.js (Vite)
Zustand (State Management + Persist)
SCSS (Custom Styling)
Lucide Icons
Backend
Node.js
Express.js
MongoDB + Mongoose
JWT Authentication
Bcrypt (Password hashing)
